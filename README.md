# Currículum Web — Alexis Daraio

Sitio web personal de una sola página que funciona como Currículum Vitae digital.

## Estructura

| Sección | Contenido |
|---------|-----------|
| **Inicio** | Presentación personal |
| **Perfil** | Descripción, cualidades e idiomas |
| **Educación** | Formación académica |
| **Contacto** | Correo electrónico con enlace directo |

## Archivos

```
curriculum-web/
├── index.html       → Toda la información en una sola página
├── css/styles.css   → Estilos del sitio
├── js/main.js       → Menú móvil
└── images/          → Imágenes
```

## Cómo ver el sitio

Abre `index.html` en tu navegador.

## Publicar en GitHub

1. Crea un repositorio en [GitHub](https://github.com/new).
2. Sube los archivos del proyecto.
3. Activa **GitHub Pages** en Settings → Pages → rama `main`.
4. Entrega el link del repositorio.

## Contacto

El botón de contacto usa un enlace `mailto:` que abre el correo del visitante con tu dirección ya cargada. No requiere servidor ni configuración extra.
